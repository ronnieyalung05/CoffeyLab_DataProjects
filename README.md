# Coffey Lab Data Projects
This repository contains a collection of projects focused on analyzing, processing, and interpreting protein expression data across multiple biological samples. It includes workflows for filtering proteins based on Gene Ontology terms, performing enrichment analyses such as over-representation analysis and gene set enrichment analysis, and visualizing complex relationships between genes and functional categories through gene-concept networks. The projects leverage both single-sample and multi-sample data, handle mapping between gene identifiers, support ranking and selection of top-expressed proteins, and provide reproducible and parallelized pipelines for efficient computation. The repository combines data engineering, functional annotation, and network visualization to enable comprehensive exploration of protein expression patterns and their biological significance.
